# ASCII Clock

A simple ASCII art clock implemented in C.

## Features

- Displays current time in ASCII art
- Customizable display options (coming soon)
- Lightweight and easy to use

## Installation

```bash
git clone https://github.com/arnabdebnath208/asciiclock.git
cd asciiclock/src/C
make
```

## Usage

```bash
cd asciiclock/src/C
./asciiclock
```
